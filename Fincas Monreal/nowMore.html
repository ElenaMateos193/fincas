Fincas Monreal

What We Do

Fincas Monreal se dedica a administrar fincas de distintas comunidades de vecinos. Podrás añadir, modificar y consultar comunidades de vecinos y sus propietarios; consulta nuestros propietarios mediante una serie de criterios como puede ser portal, portal y planta, apellidos, DNI o todos los de la comunidad.

Cuando consultes algo de nuestra base de datos podrás modificarlo.

How We Do It

Se utiliza la base de datos relacional H2 para almacenar los datos de nuestras comunidades de vecinos y sus propietarios. De esta manera se podrá modificar, añadir y consultar las entidades que implementamos.

Para ayudar a nuestro equipo de desarrollo de nuestra pequeña empresa a construir un sistema basado en JVM de manera simple, portable, rápida y flexible. Para conseguirlo nos apoyaremos en Spring Tool Suite (STS) el cual es un IDE basado en Eclipse fabricado para servir a nuestras necesidades de construir aplicaciones con Spring.

STS se apoya en Spring 4.2 y este a su vez está apoyado en un servidor web Tomcat 8 (Java EE7) que puede usar las librerias de Java Enterprise Edition 7 (Java EE7) y aquellas que están en Java SE8

Usaremos un proyecto Maven para gestionar las dependencias (librerías) que usaremos como Thymeleaf; el cual es un sistema de plantillas modernas de la parte del servidor. Además implementamos un modelo MVC (modelo-vista-controlador)donde el controlador es quien manipula los datos del modelo y se los pasa/coge a/de la vista (HTML)
